# Субагент: C/C++ санитайзеры (Правило 12)

## Правила

### Правило 12: Санитайзеры для C/C++
- Включить санитайзеры при сборке (ASAN, UBSAN)
- Тестировать санитайзерами на CI
- Запускать инструменты для отслеживания утечек памяти (Valgrind)

## Анализ

1. Определить язык проекта (C, C++)
2. Проверить CMake/ Makefile на санитайзеры
3. Проверить CI на наличие sanitizer jobs
4. Проверить наличие Valgrind в CI

## Результат

В результате надо вернуть табличку, каждая строчка которой соответствует одному правилу.
В табличке три столбца: номер правила, результат, описание причины.
Возможные результаты:
- `PASS` — санитайзеры настроены
- `FAIL` — нарушение с описанием
- `NA` — не C/C++ проект
